DistroMap API

A service that provides information about Linux distribution versions and release cycles

API Usage

This API allows you to query Linux distribution release information by product name and codename.

Available Endpoints

Get distribution info:
GET /distro/{product}/{codename}
Example: /distro/ubuntu/noble or /distro/debian/bookworm
Health check:
GET /health
Returns API status and database information
This page:
GET /
Returns this HTML documentation page

Examples

Ubuntu Noble (24.04)
/distro/ubuntu/noble - Get information about Ubuntu 24.04 LTS (Noble Numbat)

Fedora 42
/distro/fedora/42 - Get information about Fedora 42

Debian Trixie
/distro/debian/trixie - Get information about Debian 13 (Trixie)
